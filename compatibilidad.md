# Compatibilidad de script de instalacion:

en este archivo se incluyen los sistemas compatibles con el script de instalacion

## Sistemas compatibles:
* **Debian:** y todas las basadas en esta con el install.sh
* **Arch:** y todas las basadas en esta con el install.sh
* **Fedora:** y todas las basadas en esta con el install.sh
* **Void:** y todas las basadas en esta con el install.sh
* **Alpine:** y todas las basadas en esta con el install.sh
* **Android:** con el install.sh e instalando Termux
* **MacOS:** con el install.sh
* **IOS:** con el install.sh e instalando IShell
* **Windows:** con el install.bat

## Sistemas excluidos:
* **BSD:** cualquier BSD no es compatible
